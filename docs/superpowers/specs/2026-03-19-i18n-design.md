# Demo 頁多語言設計 — 自製 useLocale composable

**日期：** 2026-03-19
**狀態：** 已核准

---

## 背景

`vue-wave-visualizer` 是公開的 npm 套件，demo 頁部署於 GitHub Pages 供全球開發者瀏覽。目前 demo 頁（`demo/App.vue`）所有 UI 文字皆為繁體中文，英文使用者看不懂介面。新增繁體中文 / English 切換功能，提升國際可用性。

不使用 `vue-i18n`，理由：demo 為單頁、文字量少（約 33 個字串）、無複數/格式化需求，自製 composable 符合 YAGNI 原則且零額外依賴。

不加入日文：投資報酬率低，維護三份字典的成本大於價值。

---

## 目標

- 頁面支援繁體中文 / English 兩種語言
- 語言切換器：globe 圖示按鈕 + 點擊展開下拉選單
- 切換即時生效，無需重新整理
- 不增加 npm 依賴

---

## 架構

```
demo/
  i18n/
    index.ts       ← useLocale() composable
    zh-TW.ts       ← 繁體中文字典
    en.ts          ← 英文字典
  components/
    LangSwitcher.vue  ← globe 圖示 + 下拉選單元件
  App.vue            ← 整合 useLocale()，t('key') 取代硬編碼文字
```

## 元件設計

### `demo/i18n/index.ts`

```ts
import { ref, computed } from 'vue'
import zhTW from './zh-TW'
import en from './en'

type Locale = 'zh-TW' | 'en'
type Messages = typeof zhTW

const locale = ref<Locale>('zh-TW')
const messages: Record<Locale, Messages> = { 'zh-TW': zhTW, en }

export function useLocale() {
  const t = (key: keyof Messages) => messages[locale.value][key]
  const setLocale = (lang: Locale) => { locale.value = lang }
  return { locale, t, setLocale }
}
```

`locale` 宣告在 module 層級（非 `useLocale()` 函式內），確保全 app 共享同一個響應式狀態。

### `demo/i18n/zh-TW.ts` / `demo/i18n/en.ts`

字典物件，key 相同，value 為對應語言文字。

### `demo/components/LangSwitcher.vue`

- 無 border、無 background 的 globe SVG 圖示按鈕
- `aria-label="選擇語言 / Select language"`
- hover：顏色 `#8b949e` → `#e6edf3`
- 點擊展開下拉選單（向下展開，左對齊圖示）
- 點選單外部（`document` click listener）或按 `Escape` 鍵自動關閉
- 目前語言項目藍色高亮（`color: #58a6ff; background: #1f3a5f`）

### `demo/App.vue` 異動

- h1 與 `<LangSwitcher />` 用 `display: flex; align-items: center; gap: 10px` 排列
- 所有硬編碼中文字串改用 `t('key')` 取代
- `useLocale()` 在 `<script setup>` 頂層呼叫一次
- 含動態內容的字串處理方式：只翻譯靜態部分，動態值在呼叫端串接
  - `micError`：`t('micError') + e.message`
  - `silenceDetected`：`t('silenceDetected') + ' (' + duration + 's)'`
  - `codeTipLine2`：`t('codeTipLine2').replace('{mode}', mode.value)`（`codeTipLine1` 無動態內容）
- `usageCode` 內的按鈕文字（`micBtnInCode`、`stopBtnInCode`）跟隨語言切換
- mode 值（`waveform`、`bars` 等）為技術 API 名稱，不翻譯，維持英文顯示
